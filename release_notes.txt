# A&S for RTX 1.9.1 Patch
The wait is finally over! A new patch version is here for the Actions & Stuff 1.9.1 update. 

Here's what's new:

## 📦 What's New in the Patch (A&S 1.9.1)
* **Updated Pack Manifest**: The patch now creates a standalone `manifest.json` under the hood so the pack loads correctly, right away! (No worries — more aesthetic improvements to the manifest and texts coming in the next version!).

## 🛠️ Patcher App Updates (v2.1)
* **Quick Settings Adjust**: Easily tweak essential settings right from the main menu.
* **All Settings Adjuster (Advanced Mode)**: Added a comprehensive settings menu for power users to dive into deeper configuration options.
* **Detection Improvements**: Path detection logic has been upgraded to better find your installed packs automatically. 
* **New UI Design**: Switched the UI framework from `ttkbootstrap` to `customtkinter` for improved layout compatibility in the future.
